# 🔐 RBAC Access Control Dashboard

A cybersecurity project demonstrating **Role-Based Access Control (RBAC)** and **least-privilege access management**.

This tool simulates how enterprise identity platforms enforce access permissions based on user roles.

## Features

- User authentication
- Role-based access control
- Permission restrictions
- Admin / Analyst / Employee / Guest roles
- Access control simulation dashboard

## Security Concepts Demonstrated

- RBAC (Role-Based Access Control)
- Authorization
- Least privilege principle
- Identity security architecture
